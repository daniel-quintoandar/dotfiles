dotfiles
--------

My configuration files.

## Requirements

  * [rcm](https://github.com/thoughtbot/rcm)

## Usage

```sh
    git clone git@github.com:danieltdt/dotfiles.git ~/danieltdt.dotfiles
    rcup -d ~/danieltdt.dotfiles -x README.md -v
```
