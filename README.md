My dotfiles
-----------

My dotfiles for Ubuntu development environment.
Includes
  * bash files
  * vim files
  * git files
  * ruby files (bundler, rspec, gemrc, rdebug)
  * tmux
  * libgit2
  * pygit2
  * python files (actually, powerline extensions)
  * powerline on vim
  * powerline on bash
  * powerline on tmux

## Usage

    git clone git@github.com:danieltdt/dotfiles.git
    cd dotfiles
    ./install.rb
